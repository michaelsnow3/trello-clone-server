# Trello-Clone-Server
Server and database queries for Trello-Clone-App. Deployed with Heroku.

## Built with
- express
- knex
- postgres

## Dependencies
- bcrypt
- body-parser
- cors
- dotenv
- express
- jsonwebtoken
- knex
- pg
